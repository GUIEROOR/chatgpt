# Changelog

## 2.0.3 - 2026-07-19

- 将 `MZFinance.BadLogin.Configurator_message` 识别为 Apple Configurator 登录验证挑战。
- 修复收到该响应时验证码输入框不显示的问题。
- 首次挑战显示验证码框并自动聚焦；验证码再次失败时给出过期、错误或账号密码异常提示。
- 后端错误响应增加 `TWO_FACTOR_REQUIRED` 与 `requiresTwoFactor` 标记，前端同时保留消息文本兜底识别。
- 兼容迁移 2.0.2、2.0.1 和 2.0.0 本地核心缓存。

## 2.0.2 - 2026-07-19

- 修复 Apple ID 输入框使用 `type=email`，导致 Safari 拒绝 `+86` 手机号的问题。
- Apple ID 改为普通文本输入，支持邮箱和手机号。
- 自动规范化手机号中的空格、括号、短横线和全角 `＋`。
- 双重认证验证码默认隐藏，仅在 Apple 返回验证挑战后显示并自动聚焦。
- 验证码阶段保留当前账号和密码，登录成功后再清空敏感输入。
- 启动器缓存版本升级为 2.0.2，可从 2.0.1/2.0.0 本地核心缓存自动热更新，无需重新下载完整核心。

## 2.0.1 - 2026-07-19

- 修复 `ipa-tool.local` 在 iOS/Loon 中可能被 `*.local` bypass-tun 或 mDNS 绕过，导致面板打不开的问题。
- 面板入口迁移到 `https://apple-api.com/ipa-tool`，复用原版插件已配置的 MitM 域名。
- 保留 `/auth/*` 与 `/apps/*` 原版 API 路由；面板内部 `/api/*` 由启动器映射到本地控制路由，互不冲突。
- 启动器缓存版本升级为 2.0.1，并兼容迁移已有 2.0.0 本地核心缓存。
- 手动运行脚本时的通知链接同步改为新面板地址。

## 2.0.0 - 2026-07-19

- 合并独立 Loon 本地面板与原版 `apple-api.com`、`xiaobai.app` 兼容层。
- 全面重做移动端界面，增加深色模式、响应式布局和下载详情弹层。
- 历史版本改为索引优先、详情按需加载，降低卡顿与无效网络请求。
- 增加多账号、账号切换、收藏、搜索历史和下载准备记录。
- 增加版本缓存 TTL、状态备份恢复和缓存容量控制。
- 增加请求重试、指数退避及 HTTP/2 到 HTTP/1.1 回退。
- Timbrd 与 Bilin 版本源并发查询、合并、去重，并显示部分源故障状态。
- 增加会话状态、手动刷新和网络诊断。
- 下载改为 302 直跳 Apple CDN，避免脚本中转大文件。
- 密码默认不落盘；可通过插件参数显式允许自动刷新会话。
- 增加本地令牌和导出脱敏，避免敏感会话从本地 API 泄露。

## 1.0.0 - 2026-07-19

- 首个 Loon 本地持久化版本。
