# 太虚问道录

一款无需安装、打开浏览器即可游玩的原创大型修仙文字游戏。

## 特色

- 27 个修炼境界，从凡人直至真仙
- 灵根、宗门、装备、炼丹、炼器体系
- 六大探索区域与动态战斗
- 随机事件、主线异象、任务和成就
- 自动修炼模式
- 浏览器本地存档、导入与导出
- iPhone 与桌面端自适应界面

## 运行

直接打开 `index.html` 即可。

## GitHub Pages

进入仓库 **Settings → Pages**，将 Source 设为 `Deploy from a branch`，分支选择 `main`，目录选择 `/ (root)`。
